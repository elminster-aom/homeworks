setup module¶

Independet module for inialiting the environment, it has be ran only once (per environment) before anything else 1. It assumes that PosgresSQL is up and running and we have administratitve authorization 2. It assumes that Kafka is up and running and we have administratitve authorization 3. It enables TimescaleDB extension in PostgresSQL 4. It creates a table for storing web_health monitoring 5. It turns previous table in a hypertable partitioned, for storing monitoring metrics

setup.init_logging()¶

Initialization of basic logging to console, stdout and stderr, where log level INFO goes to stdout and anything else to stderr See: https://stackoverflow.com/questions/2302315/how-can-info-and-debug-logging-message-be-sent-to-stdout-and-higher-level-messag
